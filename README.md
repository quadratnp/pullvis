composer require laravel/breeze --dev

install breeze react
```
php artisan breeze:install react
```

install npm
```
npm install
```

install daisyUI
```
npm i daisyui
```

add daisyUI to your tailwind.config.js files:
```
module.exports = {
  //...
  plugins: [require("daisyui")],
}
```

install react-icons
```
npm install react-icons
```
